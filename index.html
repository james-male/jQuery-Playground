Toggle Content

Logic

  All show and hide functions are selected based on class names
  • Include jQuery Library (I in this example I am using a Google CDN link)
  • Default content is shown and button content hidden using an inline CSS when the page is loaded
  • On button click
    • Hide content for the buttons that were not clicked as well as the default content
    • Show content for the button that was clicked
  • On Reset click
    • Hide content from all buttons
    • Show default content

Code


HTML

// Include jQuery
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>

// Buttons
<button type="button" class="button-default">Default</button>
<button type="button" class="button-left">Left</button>
<button type="button" class="button-left">Middle</button>
<button type="button" class="button-left">Right</button>

// Content
<p class="default-content">Default Content</p>
<p class="left-content">Left Button Content</p>
<p class="middle-content">Middle Button Content</p>
<p class="right-content">Right Button Content</p>

jQuery

$(document).ready(function(){
// Button Left
 $(".button-left").click(function(){
$(".default-content").hide();
$(".middle-content").hide();
$(".right-content").hide();
$(".left-content").show();
});

// Button Middle
 $(".button-middle").click(function(){
$(".default-content").hide();
$(".left-content").hide();
$(".right-content").hide();
$(".middle-content").show();
});

// Button Right
 $(".button-right").click(function(){
$(".default-content").hide();
$(".left-content").hide();
$(".middle-content").hide();
$(".right-content").show();
});

// Default
 $(".button-default").click(function(){
$(".left-content").hide();
$(".middle-content").hide();
$(".right-content").hide();
$(".default-content").show();
});
});

Random Image Generator

Logic

  Ensure your image tag is read before your jQuery otherwise you will get the issue where the code wont be able to see what 'src' needs to be replaced
  Be sure to check that you have assigned the random number an upper limit as you wont be able to load image35.jpg if there are only 10 images in your folder in this code the number 10 defines the upper limit "var randomNum = Math.floor((Math.random()*10)+1);"
  • Include jQuery Library (I in this example I am using a Google CDN link)
  • Save a set of images with the same naming convention, in this example I have used image1.jpg, image2.jpg image3.jpg etc...
  • Define an image tag with a unique element ready to be selected. In the below example I have used the id 'random-image'
  • Randomly generate a number using Math.random() and save it to a variable
  • Update the 'src' attribute to include both the path of the folder where the images are stored and the variable that containers the random number. The result will be something that looks like this: "images/location/image2.jpg"



HTML

// Define an image tag with a unique identifier
 <img id="random-image" src="">

jQuery

// Generate a random number
var randomNum = Math.floor((Math.random()*10)+1);

// Replace the 'src' attribute
$('#random-image').attr('src', 'img/image' + randomNum + '.jpg');
